# 🎭 Name Vibe Detector 
A fun and interactive console-based Java program that analyzes your name and generates a quirky personality "vibe" based on your name's characters!

---

## 🚀 Features

- Counts vowels and consonants in your name  
- Predicts your personality vibe  
- Analyzes your name length and gives mood feedback  
- Shows a random motivational message  
- Purely based on **String manipulation concepts** — great for beginners!

---

## 💻 How to Run

1. **Clone or download** this repository.

2. Make sure you have **Java installed** on your system. If not, install it from [here](https://www.oracle.com/java/technologies/javase-downloads.html).

3. Open terminal or PowerShell inside the project folder and run:

   ```bash
   javac NameVibeDetector.java
   java NameVibeDetector

4. Enter your name and enjoy the results 🎉
